## Getting started

### Prerequisites

1. [Install Node.js](https://nodejs.org/en/download/package-manager/)

### Installation

1. Clone the repository:

```shell script
git clone https://github.com/CyberryTeam/epitech-intranet.git
cd epitech-intranet
```

2. Install Node.js packages:

```shell script
npm install
```

## Testing

Run coding style tests:

```shell script
npm run lint
```
